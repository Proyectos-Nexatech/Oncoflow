DOCUMENTO DE ESPECIFICACIONES TÉCNICAS SISTEMA WEB DE GESTIÓN Y CONTROL DE ENTREGA DE MEDICAMENTOS ONCOLÓGICOS 1. INFORMACIÓN GENERAL Nombre del Proyecto Sistema Web de Gestión de Programación, Entrega y Facturación de Medicamentos Oncológicos. Objetivo General Desarrollar un aplicativo web responsive que permita gestionar y controlar la programación diaria y semanal de entrega de medicamentos oncológicos a pacientes, incluyendo el control documental, administrativo y de facturación mensual asociado a dichas entregas. Tipo de Aplicación Aplicación Web Responsive Multiusuario Acceso por roles Arquitectura cliente-servidor 2. OBJETIVOS FUNCIONALES El sistema debe permitir: Gestionar pacientes oncológicos. Programar entregas diarias y semanales. Controlar documentación requerida para la entrega. Gestionar el inventario básico de medicamentos. Registrar y validar entregas. Controlar el proceso de facturación mensual. Generar indicadores operativos y administrativos. Gestionar usuarios y permisos. Generar reportes exportables. 3. ALCANCE FUNCIONAL El sistema incluirá los siguientes módulos: Dashboard Principal Gestión de Pacientes Programación de Entregas Gestión de Medicamentos Gestión de Entregas Gestión Documental Facturación Reportes Administración y Seguridad 4.
